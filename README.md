# Okul QR Projesi

## Kurulum
1. Bagimliliklari yukleyin:
   - `npm install`
2. Gelistirme sunucusunu baslatin:
   - `npm run dev`

## Ortam Degiskenleri
`.env.local` dosyasina EmailJS bilgilerini ekleyin:
```
NEXT_PUBLIC_EMAILJS_SERVICE_ID=service_xxx
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=template_xxx
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=public_xxx
```

## Veri Dosyalari
- `data/settings.json`
- `data/teachers.json`
